# תיעוד הגדרות אלמנטים בעמוד מוצר

מסמך זה מתאר את כל ההגדרות הזמינות לכל אלמנט בעמוד המוצר במערכת Quick Shop CRM.

## סקירה כללית

כל אלמנט בעמוד המוצר יכול להיות מותאם אישית עם הגדרות עיצוב מתקדמות, כולל:
- פונט (Font Family)
- גודל פונט (Font Size)
- עובי פונט (Font Weight)
- גובה שורה (Line Height)
- יישור טקסט (Text Alignment)
- מרווחים (Margins)

## רשימת אלמנטים והגדרותיהם

### 1. שם מוצר (Product Name)
**סוג אלמנט:** `product-name`

**הגדרות זמינות:**
- **פונט:** בחירה מרשימת פונטים זמינים
- **גודל פונט:** 8-120px (ברירת מחדל: 32px)
- **עובי פונט:** 300-900 (ברירת מחדל: 700 - עבה)
- **גובה שורה:** 0.5-3.0 (ברירת מחדל: 1.2)
- **יישור:** ימין / מרכז / שמאל (ברירת מחדל: ימין)
- **מרווח עליון:** 0px ומעלה (ברירת מחדל: 0px)
- **מרווח תחתון:** 0px ומעלה (ברירת מחדל: 16px)

**הערות:**
- זהו האלמנט הראשי של שם המוצר
- מומלץ להשתמש בפונט עבה (700-900) לגודל גדול (28-40px)

## עובי פונט (Font Weight)

הערכים הזמינים:
- **300** - דק (Light)
- **400** - רגיל (Regular/Normal)
- **500** - בינוני (Medium)
- **600** - חצי עבה (Semi-Bold)
- **700** - עבה (Bold)
- **800** - עבה מאוד (Extra-Bold)
- **900** - עבה ביותר (Black)

- **שמאל (Left)** - יישור לשמאל (LTR)

## שימוש בממשק

### פתיחת הגדרות אלמנט

1. **דרך אייקון גלגל שיניים:**
   - במצב עריכה, hover על אלמנט
   - לחץ על אייקון גלגל השיניים (⚙️) בכפתורי הבקרה

2. **דרך לחיצה על שם האלמנט:**
   - במצב עריכה, hover על אלמנט
   - לחץ על המדבקה עם שם האלמנט (בפינה הימנית העליונה)

3. **דרך דף ההתאמה האישית:**
   - פתח את דף ההתאמה האישית (`/customize`)
   - לחץ על "עיצוב אלמנטי עמוד המוצר"
   - לחץ על שם האלמנט או על אייקון גלגל השיניים

### שמירת הגדרות

- ההגדרות נשמרות אוטומטית ב-`config.style` של כל אלמנט
- השינויים נשמרים ב-database דרך ה-API endpoint: `/api/storefront/[slug]/product-page-layout`

## דוגמאות שימוש

### הגדרת שם מוצר בולט
```json
{
  "fontFamily": "var(--font-noto-sans-hebrew), sans-serif",
  "fontSize": 40,
  "fontWeight": 900,
  "lineHeight": 1.2,
  "textAlign": "right",
  "marginBottom": 24
}
```

### הגדרת תיאור קריא
```json
{
  "fontFamily": "Assistant, sans-serif",
  "fontSize": 18,
  "fontWeight": 400,
  "lineHeight": 1.8,
  "textAlign": "right",
  "marginBottom": 32
}
```

### הגדרת מחיר בולט
```json
{
  "fontFamily": "var(--font-noto-sans-hebrew), sans-serif",
  "fontSize": 32,
  "fontWeight": 700,
  "lineHeight": 1.3,
  "textAlign": "right",
  "marginBottom": 20
}
```

## הערות טכניות

- כל ההגדרות נשמרות ב-`element.config.style`
- ההגדרות מיושמות דרך inline styles ב-React
- ההגדרות עובדות עם כל הפריסות (standard, right-side, left-side, masonry, fixed)
- ההגדרות תומכות ב-RTL ו-LTR
